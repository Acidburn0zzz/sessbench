Session benchmarker

Each directive in the right pane will be executed in sequence. Each directive must be separated by a newline character. Unknown directives will be ignored.

Valid directives:

  • repeat n: Repeat the benchmark n time and report averaged results.
  • wait n: Wait n seconds before each page load.
  • clear cache: Clear the browser cache.
  • URL: A URL to benchmark.
